Nessus

2023-09-25
#Blog 

Introduction:⌗

Nessus is a widely used vulnerability assessment tool that helps organizations identify and address security issues within their networks, systems, and applications.

Installation and setup:

To install and setup Nessus, we need first the installation package found on their website by searching for Nessus download or you can use this link Reference Link, which will take you to the website in the picture bellow.

Choose which version and platform of your choice then press download. For Linux you can use the following command to install Nessus after you download the package:

sudo dpkg -i Nessus-10.6.0-debian10_amd64.deb

When the installation is successfully done, you should enable nessus.service and head to the website provided on your CLI:

/bin/systemctl start nessusd.service

When the service is enabled, we can go ahead and access the link provided to us and setup Nessus.

Choose Register for Nessus Essentials.

Register using your credentials for an activation code, then copy paste the code on the specified text box, then wait for the installation to finish.

Key Nessus components:⌗

On the left side, you can find Recourses section and under it 3 tabs:

- Policies can be used to create custom templates.
- Plugin Rules define the behavior of these plugins during a vulnerability scan.
- Terrascan can be used as static code analyzer for infrastructure as code.

New scan is used to lunch a scan on a target using various vulnerability scans.

Scan:⌗

Once you choose the type of scan you would like to perform, you need to setup the scan setting. General tab, you can setup the details for the target you want to scan. Discovery tab, you can choose the type of ports you would like to scan for. Assessment tab, you can choose the scan type, like for known web vulnerabilities or quick scan. Report tab, the type of reports you would like to be displayed. Advanced tab, can be used to setup the bandwidth of the scan.

I chose basic scan and lunched the scan after setting up the necessary settings. I received various information like port scan, website version…etc.
