Burpsuite

2023-09-25
#Blog 

Introduction:⌗

Burp Suit is a web vulnerability scanner and security testing platform developed by PortSwigger. It captures and enables manipulation of all the HTTP/HTTPS traffic between a browser and a web server.

Burp Suite Community Tools:⌗ 

• Proxy intercept and modify of requests and responses while interacting with web applications.

• Repeater allows us to capture, modify, and resend the same request multiple times. It is useful when crafting payloads or testing the functionality of an endpoint for vulnerabilities.

• Intruder allows for spraying endpoints with requests. It is commonly utilized for brute-force attacks or fuzzing endpoints.

• Decoder offers a valuable service for data transformation. It can decode captured information or encode payloads before sending them to target.

• Comparer enables the comparison of two pieces of data at either the word or byte level.

• Sequencer is typically employed when assessing the randomness of tokens, such as session cookie values or other supposedly randomly generated data. If the algorithm used for generating these values lacks secure randomness, it can expose avenues for devastating attacks.

Installation:⌗

Burp Suite installation Link, choose the proper operating system and install the zip file. On windows run the executable, and on Linux by simply running the scripting in the terminal, by giving it the necessary privelge and running it using ./and the file name.

DashBoard:⌗ 

1. Tasks: The Tasks menu allows you to define background tasks that Burp Suite will perform while you use the application. In Burp Suite Community, the default “Live Passive Crawl” task, which automatically logs the pages visited, is sufficient for our purposes in this module. Burp Suite Professional offers additional features like on-demand scans.

2. Event log: The Event log provides information about the actions performed by Burp Suite, such as starting the proxy, as well as details about connections made through Burp.

3. Issue Activity: This section is specific to Burp Suite Professional. It displays the vulnerabilities identified by the automated scanner, ranked by severity and filterable based on the certainty of the vulnerability.

4. Advisory: The Advisory section provides more detailed information about the identified vulnerabilities, including references and suggested remediations. This information can be exported into a report. In Burp Suite Community, this section may not show any vulnerabilities.

Burp Suite Proxy:⌗ 

• Intercepting Requests: Intercept options allows us to intercept requests using Burp Suite, which allows us to modify the request or analyze it. To allow the request to pass you should turn off the intercept option.

• WebSocket Support: Burp Suite also captures and logs WebSocket communication, providing additional assistance when analyzing web applications.

All captured logs can be found in the history sub-category. While intercepting requests you are able to play the man in the middle and change the request sent.

Proxy settings:⌗ 

• Response Interceptions isn't active by default unless explicitly requested on a per-request basis.

• Match and Replace section in the proxy settings enables the use of regular expressions (regex) to modify incoming and outgoing requests.

FoxyProxy:⌗

FoxyProxy is a browser extension that facilitates the management and switching of proxy servers within web browsers. It is available for popular web browsers like Mozilla Firefox and Google Chrome. Install the extension and configure it. You can configure it by going to FoxyProxy options and adding the IP and port of the Proxy then on the extension press on burp and it will be activated and it will start intercepting the websites.

Decoder:⌗

Once you find Authorisation code on the site map, you can use the code with Decoder to get the clear text of the message.

Repeater:⌗

You can use shellshock attack using repeater to change the request sent and embed an attack.
